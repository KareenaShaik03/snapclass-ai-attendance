# 📸 SnapClass AI Attendance System

An AI-powered attendance management system built with **Python**, **Streamlit**, **Face Recognition**, and **Supabase**. The application automates attendance by recognizing registered students through facial recognition, eliminating manual attendance processes while providing a secure and user-friendly interface.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/KareenaShaik03/snapclass-ai-attendance.git
```

Navigate to the project folder:

```bash
cd snapclass-ai-attendance
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root and add your credentials:

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
```

> Never upload your `.env` file to GitHub.
